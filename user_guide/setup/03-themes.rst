Themes
===========

.. warning::
	Themes are essentially custom Godot projects. This means they run custom code, and can therefore be malicious.

	Be careful where you download themes from. We recommend downloading only themes from the app and/or trusted sources.

Installing
~~~~~~~~~~

To install themes, move the theme packages inside the ```themes`` folder in RetroHub's config folder:

- **Windows:** ``C:\\Users\<user_name>\RetroHub``
- **macOS:** ``/Users/<user_name>/.retrohub``
- **Linux:** ``/home/<user_name>/.retrohub``

You can also open the theme folder from the app's settings.

After that, simple pick your theme in the settings.
