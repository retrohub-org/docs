Download
========

RetroHub can be downloaded from `GitHub releases <https://github.com/retrohub-org/retrohub/releases>`_.

The current latest release is `v0.0.2-alpha <https://github.com/retrohub-org/retrohub/releases/tag/v0.0.2-alpha>`_.

RetroHub is available for **Windows**, **macOS** and **Linux**.

Installing
~~~~~~~~~~

Windows
^^^^^^^

Windows releases work like portable applications. Extract to a folder of your choice and run the main executable `RetroHub.exe`

macOS
^^^^^

macOS releases as an app bundle. Launch `Retrohub.App`

Linux
^^^^^

Linux releases work like portable applications. Extract to a folder of your choice and run the main executable `RetroHub`.

.. note::
	You may need to mark it as executable.
