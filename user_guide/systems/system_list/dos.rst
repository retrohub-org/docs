DOS (PC)
========

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``dos``
- **Type:** Computer
- **Supported extensions:** 16
	- .exe
	- .com
	- .bat
	- .zip
	- .dosz
	- .iso
	- .cue
	- .ins
	- .img
	- .ima
	- .vhd
	- .jrc
	- .tc
	- .m3u
	- .m3u8
	- .conf

Notes
~~~~~

There are no special notes for this system. Games should work out of the box.
