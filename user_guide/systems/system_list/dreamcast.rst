Sega Dreamcast
==============

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``dreamcast``
- **Type:** Console
- **Supported extensions:** 10
	- .cdi
	- .gdi
	- .chd
	- .cue
	- .bin
	- .elf
	- .lst
	- .dat
	- .m3u
	- .zip

Notes
