NEC PC Engine
=============

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``pcengine``
- **Type:** Console
- **Supported extensions:** 9
	- .pce
	- .sgx
	- .cue
	- .ccd
	- .chd
	- .iso
	- .img
	- .bin
	- .zip
