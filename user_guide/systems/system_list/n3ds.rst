Nintendo 3DS
============

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``n3ds``
- **Type:** Modern Console
- **Supported extensions:** 7
	- .3ds
	- .3dsx
	- .elf
	- .axf
	- .cci
	- .cxi
	- .app

Notes
