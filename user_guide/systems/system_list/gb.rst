Nintendo Game Boy
=================

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``gb``
- **Type:** Console
- **Supported extensions:** 8
	- .gb
	- .gbc
	- .bin
	- .rom
	- .dmg
	- .cgb
	- .sgb
	- .zip

Notes
~~~~~

There are no special notes for this system. Games should work out of the box.
