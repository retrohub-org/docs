Nintendo Game Boy Color
=======================

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``gbc``
- **Type:** Console
- **Supported extensions:** 8
	- .gb
	- .gbc
	- .bin
	- .rom
	- .dmg
	- .cgb
	- .sgb
	- .zip

Notes
