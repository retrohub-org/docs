Nintendo Entertainment System
=============================

Basic Information
~~~~~~~~~~~~~~~~~
- **Short name:** ``nes``
- **Type:** Console
- **Supported extensions:** 5
	- .nes
	- .fds
	- .unf
	- .unif
	- .zip

Notes
~~~~~

There are no special notes for this system. Games should work out of the box.
